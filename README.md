# Projet PL Capteurs

## Installer le venv

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

## Lancer les instances

```powershell
python main.py instances/fichier-exemple.txt
python main.py instances/moyen_test_2.txt
python main.py instances/moyen_test_3.txt
python main.py instances/gros_test_1.txt
python main.py instances/maxi_test_1.txt
```
